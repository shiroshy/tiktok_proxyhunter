TikTok Proxy Hunter для Windows

Запустите TikTokProxyHunter.exe, пройдите первичную настройку и нажмите
«Начать быстрый поиск». Результаты сохраняются в выбранной папке и доступны
на страницах «Результаты» и «История».

Chromium для необязательной проверки воспроизведения устанавливается отдельно
из раздела «Настройки». GeoIP MMDB также не входит в поставку.

Безопасность:
- не вводите через бесплатный прокси пароли, токены или платёжные данные;
- сначала используйте только чистый browser context и не основной аккаунт;
- приложение не меняет системные proxy-настройки и не обходит CAPTCHA;
- статус «Рекомендован» не гарантирует долговременную работу публичного прокси.

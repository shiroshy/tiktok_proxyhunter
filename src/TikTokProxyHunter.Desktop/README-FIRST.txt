TikTok Proxy Hunter — первый запуск

1. Запустите TikTokProxyHunter.exe.
2. Пройдите короткую первичную настройку.
3. Нажмите «Начать быстрый поиск».
4. Дождитесь результата или безопасно отмените поиск и продолжите позже.
5. Копируйте только проверенные прокси из раздела «Результаты».

Публичные прокси нестабильны и потенциально небезопасны. Не вводите через них
пароли, токены или платёжные данные. Приложение не меняет системные proxy-настройки,
не входит в TikTok и не обходит CAPTCHA.

Chromium для проверки воспроизведения устанавливается отдельно только после вашего
явного подтверждения. GeoIP MMDB не обязательна и не включена в поставку.
